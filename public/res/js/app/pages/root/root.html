What is included?

  • Server with nodeJS and ExpressJS
  • MongoDB integration with Mongoose layer
  • Single Page application starter kit using AngularJS, RequireJS
  • SEO support by rendering the pages in server side using Zombie framework. Reference: Seo for Single Page Applications
  • r.js optimizer integration for whole project using gulp
  • r.js support for CSS also using require-css. You can load CSS on demand for the pages are you are in, instead of loading all CSS.
  • Google cloud storage CDN integration for static resources and dynamic assets like user uploaded images
  • Complete social Logins using hello.js in client side
  • Environment specific configurations
  • Separate logging for DB queries, http requests, error etc, using winston and morgan
  • JSONP support for data
  • Auth check for client side routes. example: "Edit your Profile" page. Reference: Protecting routes in AngularJS
  • Dynamic meta content like page title, meta description, meta content for SEO. Reference: StackOverflow
